Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Header text is created using the h1-h6 elements
  • Unordered lists are created using the ul element
  • Listed points on an unordered list are created using the li element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS can be used to create styles for websites and apply rules/changes to various elements
  • Class selectors can be applied in the "section" of HTML code in order to add attributes to that element.
  • The "link" element is used in the head of the code using rel and href to link files together, such as a CSS files to an HTML files

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commands always start with git
  • Branches are copies of the main code that can be edited in order to not edit the main code and allow for easy integration
